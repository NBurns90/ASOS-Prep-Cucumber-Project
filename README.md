# ASOS-Prep-Cucumber-Project
Java Cucumber Project - ASOS Website

## Purpose
The purpose of this project will be to test a handful of ASOS' website features using Cucumber in Java.

### Tech
* Maven
* Java
* Selenium
* Cucumber

## What's Being Tested
